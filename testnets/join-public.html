# Validators

Requires desmos executables

To join the public testnet you must have desmos installed.

To become a testnet validators, the mainnet instructions apply:

  1. Create a full node.
  2. Become a validator

The only difference is the Desmos software version and genesis file. See the testnet repo (opens new window) for information on testnets, including the correct version of Desmos to use and details about the genesis file.
